3.1 geom_area

3.1.1 Package

ggplot2 (Wickham 2016)

3.1.2 Description

Create an area plot. This cover the space between x-axis and line that connects the data points.

3.1.3 Understandable aesthetics

alpha, colour, linetype, linewidth

3.1.4 The statistical transformation to use on the data for this layer

stat_align

3.1.5 See also

geom_line, geom_ribbon

3.1.6 Example

a1 <- worldbankdata |>
  filter(Country == "Bangladesh") |> 
  filter(Year >= 2013 & Year <= 2021) |>
  ggplot(aes(x=Year, y=Electricity)) + 
  geom_area(alpha=0.5) + 
  theme(axis.text.x = element_text(angle = 90, vjust = 0.5, hjust=1)) + 
  scale_x_continuous(breaks = 2013:2021) + 
  labs(title = "a1: geom_area only")

a2 <-  worldbankdata |>
  filter(Country == "Bangladesh") |> 
  filter(Year >= 2013 & Year <= 2021) |>
  ggplot(aes(x=Year, y=Electricity)) + 
  geom_area(alpha=0.5) + 
  theme(axis.text.x = element_text(angle = 90, vjust = 0.5, hjust=1)) + 
  scale_x_continuous(breaks = 2013:2021)  + 
  geom_point(col="red") + 
  labs(title = "a2: geom_area \n and geom_point")
a1 | a2

3.2 geom_abline

3.2.1 Package

ggplot2 (Wickham 2016)

3.2.2 Description

Description Draw a straight line (\(Y=mX+c\)) for a given slope (\(m\)) and intercept (\(c\)).

3.2.3 Understandable aesthetics

Unlike most other geoms, geom_abline does not depend on the x and y variables that we map for the main plot. geom_abline has its own independent characteristics: intercept and slope.

3.2.4 Statistics layer(s)

None (geom_abline is independent from the data layer.)

3.2.5 See also:

geom_point, geom_vline, geom_hline

3.2.6 Example

a1 <- ggplot(worldbankdata, aes(y = Cooking, x=Electricity)) + 
  geom_abline(intercept = 0, slope = 1) + 
  labs(title="a1: geom_abline only") +
  theme(aspect.ratio = 1)
a2 <- ggplot(worldbankdata, aes(y = Cooking, x=Electricity)) + 
  geom_abline(intercept = 0, slope = 1) + 
  geom_point() + 
  labs(title = "a2: geom_abline \n and geom_point") +
  theme(aspect.ratio = 1)
a1 | a2

3.3 geom_alluvium

3.3.1 Package

ggalluvial(Brunson and Read 2019; Brunson 2020)

3.3.2 Description

Create alluvial plot. An alluvial plot is a type of diagram that is particularly useful for visualizing categorical data and the flow or transition between different categorical variables over multiple stages or categories

3.3.3 Understandable aesthetics

x, y, ymin, ymax, alpha, colour, fill, linetype, size, group (group is used internally; arguments are ignored)

3.3.4 Statistics layer(s)

alluvium

3.3.5 See also

geom_stratum, geom_flow, geom_lode

3.3.6 Example

library(ggalluvial)
freq.table <- worldbankdata |>
  select(Country, Region, Year, Income) |>
  filter(Year > 2015) |>
  group_by(Region, Year, Income) |>
  summarise(n = n()) |>
  drop_na()

3.4 geom_arc

3.4.1 Package

ggforce(Pedersen 2022)

3.4.2 Description

Draw a circle or a segment of a circle.

3.4.3 Understandable aesthetic

required aesthetics

x0 - starting coordinate of x-axis , y0 - starting coordinate of x-axis, r - radius, start, end

optional

color, linewidth, linetype, alpha, lineend

3.4.4 The statistical transformation to use on the data for this layer

stat_arc

3.4.5 See also

geom_arc2, geom_arc_bar

3.4.6 Example

library(ggforce)
ggplot() + 
  geom_arc(aes(x0=0, y0=0, r=8, start=1, end=8)) + 
  geom_arc(aes(x0=0, y0=0, r=8, start=1, end=5), col="red",size=2) + theme(aspect.ratio = 1)

3.5 geom_arc_bar

3.5.1 Package

ggforce(Pedersen 2022)

3.5.2 Description

To draw pie chart and donut chart defining centre point, a radius and a start and end angle.

3.5.3 Understandable aesthetic

required aesthetics

x0 - starting coordinate of x-axis , y0 - starting coordinate of x-axis, r - radius, start, end

optional

color, linewidth, linetype, alpha, lineend

3.5.4 The statistical transformation to use on the data for this layer

stat_arc

3.5.5 See also

geom_arc, geom_arc_bar

3.5.6 Example

df <- data.frame(
    state = c('A', 'B', 'C', 
              'D', 'E'),
    focus = c(0.2, 0, 0, 0, 0),
    start = c(0, 1, 2, 3, 4),
    end = c(1, 2, 3, 4, 2*pi),
    amount = c(4, 3, 1, 2, 5),
    stringsAsFactors = FALSE
)
ggplot(df) + 
    geom_arc_bar(aes(x0 = 0, y0 = 0, r0 = 0, r = 1, amount = amount, 
                     fill = state, explode = focus), stat = 'pie') + 
  theme(aspect.ratio = 1)
